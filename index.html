Samsung Internet friendly

유튜브 백그라운드 재생용 미니 플레이어

영상을 한 번 직접 재생한 뒤 홈 화면으로 나가거나 화면을 끄면, Samsung Internet의 미디어 처리 방식에 따라 오디오가 계속 재생될 수 있습니다.

플레이어 준비 중…

사용 방법

  1. Samsung Internet 모바일앱에서 이 페이지를 엽니다.
  2. YouTube 링크를 넣고 불러오기를 누릅니다.
  3. 영상이 보이면 재생을 한 번 직접 누릅니다.
  4. 홈 화면으로 나가거나 화면을 꺼서 백그라운드 재생 여부를 확인합니다.

팁

  • 자주 듣는 영상 링크를 넣어두면 다음 접속 때 자동으로 기억됩니다.
  • 미니모드를 누르면 작은 플레이어로 화면에 유지할 수 있습니다.
  • 화면을 다시 열었을 때 멈춰 있으면 재생을 한 번 더 눌러주세요.
  • 다른 영상을 듣고 싶으면 새 YouTube 링크를 넣고 불러오기를 누르면 됩니다.
